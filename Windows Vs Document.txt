

Window object : 
1. It is the top most object and outermost element of the object hierarchy.
2. Screen is the window property that holds information of browser screen.
3. The window object represents a window/tab containing a DOM document.

Document object : 
1. Each HTML document that gets loaded into a window becomes a document object.
2. The document contains the contents of the page. 
3. Using document object, JavaScript can modify, add and delete the HTML elements, 
   attributes CSS styles in the page.
